# CSS Advanced - Techium Project

This project is part of the Holberton School web front-end curriculum. It focuses on advanced CSS techniques to style a homepage for Techium, a digital company.

## Files

- `index.html` - Main HTML file
- `styles.css` - CSS stylesheet
- `images/` - Directory containing all images

  - `logo-white.png` - White logo
